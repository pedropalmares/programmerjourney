Integrate Blazor into Razor Pages and MVC

Fri, Jul 10, 2020 by Pedro

I’ve been looking at Blazor components for some time and I was looking for a way to integrate it into my existing ASP.NET Core Web applications. It’s very easy!

  1. In the layout file _Layout.cshtml add this <base> tag to the <head> element:
<base href="~/" />
  1. Also in the layout file _Layout.cshtml add this <script> tag before the closing <body> tag:
<script src="_framework/blazor.server.js"></script>
  1. Add an _Imports.cshtml to your Pages folder.
@using System.Net.Http
@using Microsoft.AspNetCore.Authorization
@using Microsoft.AspNetCore.Components.Authorization
@using Microsoft.AspNetCore.Components.Forms
@using Microsoft.AspNetCore.Components.Routing
@using Microsoft.AspNetCore.Components.Web
@using Microsoft.JSInterop
@using MyNamespace    // Change to your namespace

Put your namespace in the highlighted row
