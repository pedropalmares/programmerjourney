Adding comments to a Hugo blog with Disqus

Wed, Jun 17, 2020 by Pedro

In this post I’ll show you how to add comments to a Hugo blog with Disqus.

Signup to Disqus

The first step is to create an account on Disqus and add your site to the account. The basic plan is free to use.

Add Disqus Short Name

Then we need to add the Disqus Short Name to your blog config.toml

disqusShortname = "yourshortname"

Done! It’s very easy!

For more details read the official documentation.
